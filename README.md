## Features

- Add, remove, and list shopping items.
- Random item generator.
- Add custom items.
- Local saving to file.

## Environment

- Python (written in ED 1.14).
- Amiga Workbench 1.3.
- Python3 in Windows 10 PowerShell to test the program.
